# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-XX

### Added
- Initial release of OpenToWorkBadge component
- Customizable badge text, size, colors, and positioning
- SVG text path following circular curve
- Gradient overlay with customizable colors and angles
- TypeScript support with full type definitions
- CSS-in-JS styling with CSS variable support
- Optional fade-in animation
- Absolute positioning support

### Features
- Fully parameterized component with sensible defaults
- No external dependencies (except React)
- Accessible with required alt text
- Responsive design support
- Easy integration with existing projects
